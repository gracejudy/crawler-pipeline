# crawler-pipeline

구매대행 자동화를 위한 크롤러/등록 파이프라인 프로젝트.

## Project Structure
- `src/` : 실제 코드 (목표 구조)
- `docs/` : 운영/설계/실행 문서
- `.env.example` : 환경변수 예시
- `CHANGELOG.md` : 변경 이력

## Quick Start
```bash
cd backend
npm install
node index.js
```

> 현재 실행 코드는 `backend/`에 있으며, `src/` 중심 구조로 점진 마이그레이션 중입니다.
